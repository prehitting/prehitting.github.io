Redis命令_字符串(String)

本文最后更新于：4 天前

String命令

  1. 获取(设定)值
set (key) (value)
get (key)
getrange (key) start end #范围获取
getset (keyvalue) #获取并设定，返回原值
  1. 对 key 所储存的字符串值，获取（设定）指定偏移量上的位(bit)。
getbit key offset
setbit key offset value
  1. 获取一个或多个给定key的值
mget key1[key2..]
  1. 更改
incr key #将 key 中储存的数字值增
incrby key increament#将 key 所储存的值加上给定的增量值（increment）
incrbyfloat key increament#将 key 所储存的值加上给定的浮点增量值
decr key #将 key 中储存的数字值减一
decrby key decrement#key 所储存的值减去给定的减量值
append key value#如果 key 已经存在并且是一个字符串， APPEND 命令将指定的 value 追加到该 key 原来值（value）的末尾
