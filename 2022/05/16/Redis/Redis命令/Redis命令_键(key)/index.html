Redis命令_键(key)

本文最后更新于：14 天前

键（key）命令

  1. 设置键值对
set (keyname) (keyvalue)
  1. 删除键
del (key)
  1. 序列化给定key，并返回被序列化的值
dump (key)
  1. 检查给定key是否存在
exists key
  1. 给key设置过期时间(以秒为单位)
expire (key) (seconds)
expire (key) (timestamp)
EXPIREAT 的作用和 EXPIRE 类似，都用于为 key 设置过期时间。 不同在于 EXPIREAT 命令接受的时间参数是 UNIX 时间戳(unix timestamp)。
pexpire (key) (milliseconds)
设置key过期时间(以毫秒为单位)
PEXPIREAT (key) (milliseconds-timestamp)
设置 key 过期时间的时间戳(unix timestamp) 以毫秒计
  1. 查找所有符合给定模式(pattern)的key
keys pattern

实例

keys *

列出所有键

  1. 移动key到指定数据库
move key db
  1. 移除key期间，key将长久保持
persist (key)
  1. 返回key剩余的过期时间
pttl (key)
#以毫秒为单位
ttl (key)
#以秒为单位
  1. 从数据库中随机返回一个key
randomkey
  1. 修改key名称
rename (key) (newkey)
  1. 仅当newkey不存在时，将key改名为newkey
renamenx (key) (newkey)
  1. 返回key所存储的值的类型
type key
  1. 迭代数据库中的数据库键
scan cursor [MATCH pattern] [COUNT count]
